# Список источников для домашнего задания N1

1. auto.ru
Объекты - автомобили. 
Атрибуты: марка, модель, год выпуска, коробка, пробег, цена

2. drom.ru
Дополнить набор данных с предыдущего сайта
Объекты автомобили. 
Атрибуты: марка, модель, год выпуска, коробка, пробег, цена

3. avito.ru
Например собрать данные по предлагаемой недвижимости. 
Атрибуты: тип недвижимости, район, площадь, цена, этажность

4. airbnb.com
Информация по предлагаемой внаем недвижимости
Атрибуты: тип недвижимости, район, площадь, количество мест, стоимость проживания

5. udemy.com
Объекты - обучающие курсы. 
Атрибуты: название, цена, навыки, продолжительность, количество студентов, оценка, отзывы

6. Google places api
Объекты - организации или достопримечательности, принадлежащие определенной геолокации.
Атрибуты: места, адрес, оценка, отзывы, время работы

7. indeed.com
Объекты - информация о вакансиях
Атрибуты: страна, город, профессия, навыки, уровень зарплаты, график работы

8. sports.ru
Объекты - игроки либо команды
Атрибуты виды спорта, достижения, страны, города, матчи

9. instagram.com
Объекты - пользователи сети
Атрибуты: различные персональные данные, контакты, геолокации

10. facebook.com
Объекты - пользователи сети
Атрибуты: различные персональные данные, контакты, геолокации, приложения
